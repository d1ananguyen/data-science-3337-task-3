# ☀️ Solar Flare Analysis — COSC 3337 Task 3

This repository contains the complete analysis, source code, and visuals for **Solar Flare Data Analysis (Task 3)**, based on the **RHESSI Solar Cycle 24 dataset (2008–2018)**.  
The project focuses on understanding solar flare activity through statistical, temporal, and spatial modeling methods.

---

## ⚙️ Setup Instructions

Before running any notebooks, make sure to **install all required dependencies** using the provided `requirements.txt` file.

```bash
# Clone this repository
git clone https://github.com/<your-username>/solar-flare-analysis.git
cd solar-flare-analysis

# (Optional) Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate  # Mac/Linux
.venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt
```

---

## ▶️ Running the Code

Each activity (1–5) corresponds to a self-contained Jupyter notebook.  
Run each notebook **from top to bottom** in order to reproduce results without errors.  
All **visuals, results, and generated images** are stored in the **respective folders** for each activity.
